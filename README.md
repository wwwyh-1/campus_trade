# 校园闲置交易平台后端系统

## 一、项目简介

本项目是一个基于 C++ Drogon 框架开发的校园闲置交易平台后端系统，面向校园二手商品交易场景，提供用户注册登录、商品发布与查询、订单创建与管理等核心功能。

项目采用 RESTful API 设计，使用 MySQL 进行数据持久化，Redis 缓存热点商品详情，并通过数据库事务与库存原子扣减机制保证订单创建过程中的数据一致性，避免并发场景下出现超卖问题。

---

## 二、技术栈

- 编程语言：C++
- Web 框架：Drogon
- 数据库：MySQL
- 缓存：Redis
- 构建工具：CMake
- 测试工具：curl、Shell 脚本
- 开发环境：Ubuntu 20.04、VS Code

---

## 三、项目功能

### 1. 用户模块

- 用户注册
- 用户登录
- 用户名重复校验
- 用户信息持久化到 MySQL

### 2. 商品模块

- 发布商品
- 查询商品列表
- 查询商品详情
- 修改商品信息
- 下架商品
- 商品详情 Redis 缓存
- 商品更新 / 下架后删除 Redis 缓存

### 3. 订单模块

- 创建订单
- 查询用户订单
- 取消订单
- 创建订单时扣减库存
- 使用事务保证库存扣减与订单生成的一致性

### 4. 并发控制

- 使用 `UPDATE ... WHERE stock > 0` 实现库存原子扣减
- 使用 Shell 脚本模拟并发下单
- 验证高并发场景下不会出现超卖问题

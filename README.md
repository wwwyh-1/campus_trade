# 校园闲置交易平台后端系统

## 一、项目简介

本项目是一个基于 C++ Drogon 框架开发的校园闲置交易平台后端系统，面向校园二手商品交易场景，提供用户注册登录、商品发布与查询、订单创建与管理、秒杀下单等核心功能。

项目采用 RESTful API 设计，使用 MySQL 进行数据持久化，引入 Redis 缓存热点商品详情，并基于 Redis Lua 脚本实现秒杀库存原子预减。订单创建过程中结合 MySQL 事务与库存原子扣减机制，保证库存扣减与订单生成的一致性，避免并发下单和秒杀场景中的超卖问题。

## 三、项目功能

### 1. 用户模块

- 用户注册
- 用户登录
- 用户名重复校验
- 用户信息持久化到 MySQL

### 2. 商品模块

- 发布商品
- 查询商品列表
- 查询商品详情
- 修改商品信息
- 下架商品
- 商品详情 Redis 缓存
- 商品更新 / 下架后删除 Redis 缓存

### 3. 订单模块

- 创建订单
- 查询用户订单
- 取消订单
- 创建订单时扣减库存
- 使用 MySQL 事务保证库存扣减与订单生成的一致性

### 4. 秒杀模块

- 秒杀下单接口
- Redis 秒杀库存初始化
- Redis Lua 脚本原子预减库存
- 秒杀成功后删除商品详情缓存
- MySQL 事务创建订单
- Redis 预减库存失败回滚处理

### 5. 并发控制

- 使用 `UPDATE ... WHERE stock > 0` 实现 MySQL 库存原子扣减
- 使用 Redis Lua 脚本保证秒杀库存判断与扣减的原子性
- 使用 Shell 脚本模拟并发下单与秒杀请求
- 验证并发场景下订单数量与库存扣减结果一致，避免超卖
